# Recommender Systems

Collection of recommendation engines built using collaborative filtering and content-based approaches.

## Projects

### Movie Recommender System
- Content-based filtering using NLTK
- Genre and metadata analysis
- **Live Demo:** [movie-recommendation-95tu.onrender.com](https://movie-recommendation-95tu.onrender.com)

### Book Recommender System
- Collaborative filtering approach
- User-based recommendations

## Tech Stack
Python | NLTK | Pandas | NumPy | Flask | Scikit-learn

## Installation
```bash
pip install nltk pandas numpy flask scikit-learn
```
